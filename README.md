# 음성인식 테이블오더

## 개요

### 프로젝트명

> 음성인식 테이블오더

### 프로젝트 소개

> 사용자가 터치식 주문과 음성인식 주문을 선택하여 사용 가능한 테이블오더입니다.  
> 테이블오더 조작이 불편한 사용자도 음성으로 편리하게 주문할 수 있도록 개발했습니다.


> MVVM 패턴을 사용하여 테이블오더를 개발하였고  
> 테이블오더에서 주문한 데이터를 TCP를 통해 주방 KDS로 전송합니다.

### 기술 스택

> WPF, C#, MSSQL
